# Магистерская выпускная работа "Обучение модели определения риска развития аутизма у детей на основе анализа снимков МРТ головного мозга"

В данной работе содержатся все скрипты использованные для выполнения
магистерской выпускной работы. В процессе выполнения работы было обучено
3 модели: Модель на оригинальных данных, на расширенных путём аугментации,
и модель обученная путём дообучения модели VGG-16, а также их сравнительная
оценка.

## Файлы проекта

### TrainInitialModel.py
Файл со скриптом обучения модели на оригинальных входных данных

### TrainAugmentedModel.py
Файл со скриптом обучения модели на аугментированных данных

### TrainVGGModel.py
Файл со скриптом дообучения модели VGG-16

### TestingModels.py
Файл со скриптом анализа и оценки обученных моделей с целью выявления наилучшей

### Utils.py
Файл с различными функциями, используемыми в других файлах проекта

### .env
Файл содержащий переменную окружения для директории с изображениями для обучения.
Входных изображения для обучения должны быть заранее разделены на изображения с
отклонениями и без и расположены в соответствующих поддиректориях. (Метки yes и no соответственно)
